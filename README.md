## Installation

### MyDE Repository

For compiled packages, add [mydehq Repository](https://mydehq.github.io/arch-repo/) to you system

### AUR

To install `catnap`, you can use an AUR helper like `yay` or `paru`. Here are the commands for both:

- Using `yay`:

```bash
yay -S catnap
```

- Using `paru`:

```bash
paru -S catnap
```
